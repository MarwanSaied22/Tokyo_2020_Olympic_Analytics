## 📋 Project Overview
This project analyzes the Tokyo 2020 Summer Olympics dataset. The goal was to clean, model, and visualize data related to teams, and medal standings to identify trends across different participating nations.

## 🚀 Key Features & Insights
* 🥇 **Medal Standings:** A dynamic leaderboard showing Gold, Silver, and Bronze counts.
* 📈 **Performance Metrics:** Visualization of team efficiency.
* 🔍 **Interactive Filters:** Capability to slice data by Country.

## 🛠️ Technical Workflow
1. **Data Cleaning (Power Query):** 
   * Handled missing values and standardized naming conventions for disciplines.
   * Performed data transformations like unpivoting and merging to create a clean schema.
2. **Data Modeling:** 
   * Established relationships between Team and Medal tables.
3. **DAX Measures:**
   * Created custom measures for total medals, weighted rankings, and participation percentages.
